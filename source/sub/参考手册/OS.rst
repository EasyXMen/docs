|                 | -  OSMEMORY_IS_WRITEABLE（Access）                 |
|                 |    不为0表示有可写权限                             |
|                 |                                                    |
|                 | -  OSMEMORY_IS_EXECUTABLE（Access）                |
|                 |    不为0表示有可执行权限                           |
|                 |                                                    |
|                 | -  OSMEMORY_IS_STACKSPACE（Access）                |

-----

OS在AUTOSAR
OS标准基础上将IOC配置界面划分出了两部分：1.OsIocBottom配置界面，2.
OsIocUpper配置界面。

OsIocBottom配置界面为AUTOSAR
OS标准配置参数，OsIocUpper配置界面为普华定制出的配置参数。
